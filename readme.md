# OpenCV Face Detection

A beginner computer vision project using Python and OpenCV that detects faces in real time using a webcam.

## Features
- Real-time webcam feed
- Face detection using Haar Cascades
- Bounding boxes around detected faces
- Press 'q' to quit

## Technologies Used
- Python
- OpenCV

## Installation

Open terminal in the project folder and run:

```bash
pip install opencv-python
```

## Run The Project

In terminal:

```bash
python main.py
```

## Controls

| Key | Action |
|---|---|
| q | Quit program |
